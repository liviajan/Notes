Colors ERD

ERD Description

Table PrimaryColor

  • Table lists 4 primary colors
  • Table has 2 attributes: Id (INT) and Color (VARCHAR)
  • Each color is identified by unique number(Id) which is the primary key of the table

Table Shades

  • Table lists color shades
  • Table has 3 attributes: Id (INT), PrimaryColorId (INT), ColorName (VARCHAR)
  • Id is the PK of the table, PrimaryColorId (FK) references PrimaryColor table with relation one to many
